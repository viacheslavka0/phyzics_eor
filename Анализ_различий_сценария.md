#### 2. Адаптивный подбор задач на основе оценки

**Что нужно сделать:**
- Создать алгоритм выбора следующей задачи:
  - Если оценка 1-2 → выбрать задачу с `difficulty` выше текущей
  - Если оценка 3 → оставить задачу с тем же `difficulty`
  - Если оценка 4-5 → выбрать задачу с `difficulty` ниже или добавить подсказки
- Реализовать endpoint для получения следующей задачи: `GET /api/session/<id>/next_task/`
- Обновить фронтенд для динамического обновления списка задач

**Алгоритм:**
```python
def get_next_task(session, last_task_rating):
    """Подбор следующей задачи на основе оценки предыдущей"""
    last_task = session.task_attempts.last().task
    current_difficulty = last_task.difficulty
    
    if last_task_rating <= 2:  # Очень легко / Легко
        # Повышаем сложность
        next_difficulty = min(5, current_difficulty + 1)
    elif last_task_rating == 3:  # Средне
        # Оставляем ту же сложность
        next_difficulty = current_difficulty
    else:  # 4-5: Сложно / Очень сложно
        # Понижаем сложность
        next_difficulty = max(1, current_difficulty - 1)
    
    # Выбираем задачу с нужной сложностью, которую еще не решали
    solved_task_ids = session.task_attempts.values_list('task_id', flat=True)
    next_task = Task.objects.filter(
        ks=session.ks,
        difficulty=next_difficulty
    ).exclude(id__in=solved_task_ids).first()
    
    return next_task
```

## 💡 РЕШЕНИЕ ДЛЯ ПРОВЕРКИ ТИПОВОЙ ЗАДАЧИ

**Проблема:** Как проверить формулировку типовой задачи, если это свободный текст?

**Решение (самопроверка):**
1. Ученик вводит свою формулировку в текстовое поле
2. Нажимает "Проверить формулировку"
3. Система показывает эталонную формулировку рядом с введенной
4. Ученик САМ сравнивает и понимает, правильно ли он сформулировал
5. Нет автоматической проверки - только самопроверка
6. Ученик может продолжить в любом случае (даже если формулировка не совпадает)

**Альтернативное решение (нечеткое сравнение):**
- Можно добавить опциональную проверку по ключевым словам
- Но это не обязательно - самопроверка достаточна для обучения
