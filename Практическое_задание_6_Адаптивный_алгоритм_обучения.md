### 1. Модели для отслеживания прогресса

#### 1.1. Модель LearningSession

Создайте модель для отслеживания сессии обучения:

**Требования:**
- Связь с пользователем и курсом/системой знаний
- Поля для отслеживания прогресса:
  - `current_stage` — текущий этап обучения
  - `tasks_solved_count` — количество решённых задач
  - `tasks_correct_count` — количество правильных ответов
  - `wrong_attempts_in_row` — количество ошибок подряд
  - `score_percent` — процент правильных ответов
- Поля для времени: `started_at`, `finished_at`

**Пример:**
```python
class LearningSession(models.Model):
    """Сессия обучения ученика"""
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    course = models.ForeignKey(Course, on_delete=models.CASCADE)  # или ваша модель
    
    # Прогресс
    current_stage = models.CharField(
        max_length=50,
        default="start",
        choices=[
            ("start", "Начало"),
            ("solving", "Решение задач"),
            ("completed", "Завершено"),
        ]
    )
    
    # Статистика
    tasks_solved_count = models.PositiveIntegerField(default=0)
    tasks_correct_count = models.PositiveIntegerField(default=0)
    wrong_attempts_in_row = models.PositiveIntegerField(default=0)
    score_percent = models.FloatField(default=0)
    
    # Время
    started_at = models.DateTimeField(auto_now_add=True)
    finished_at = models.DateTimeField(null=True, blank=True)
    
    class Meta:
        verbose_name = "Сессия обучения"
        verbose_name_plural = "Сессии обучения"
    
    def __str__(self):
        return f"{self.user.username} — {self.course.title}"
    
    def calculate_score(self):
        """Вычислить процент правильных ответов"""
        if self.tasks_solved_count > 0:
            self.score_percent = (
                self.tasks_correct_count / self.tasks_solved_count * 100
            )
        return self.score_percent
```

#### 2.1. Алгоритм на основе последовательности ошибок (обязательно)

**Требования:**
- Отслеживание `wrong_attempts_in_row`
- При правильном ответе → сброс счётчика
- При неправильном ответе → увеличение счётчика
- Если 2 ошибки подряд → уменьшение сложности (или переход на другой этап)

**Реализация в API:**

#### 2.3. Алгоритм "лестницы" (Staircase) (опционально, +3 балла)

**Принцип:** При правильном ответе увеличиваем сложность, при неправильном — уменьшаем.

**Требования:**
- Отслеживание текущей сложности
- При правильном ответе → сложность +1
- При неправильном ответе → сложность -1
- Ограничения: минимум 1, максимум 5

**Реализация:**
```python
def get_next_task_difficulty(session):
    """Определить сложность следующей задачи по алгоритму лестницы"""
    last_attempt = TaskAttempt.objects.filter(
        session=session
    ).order_by('-created_at').first()
    
    if not last_attempt:
        return 3  # Начальная сложность
    
    current_difficulty = last_attempt.task.difficulty
    
    if last_attempt.is_correct:
        return min(current_difficulty + 1, 5)
    else:
        return max(current_difficulty - 1, 1)
```

Создайте эндпоинт для создания/получения сессии:

**Требования:**
- `GET /api/session/current/?course_id=X` — получить текущую сессию или создать новую

**Пример:**
```python
@action(detail=False, methods=["get"])
def current(self, request):
    """Получить текущую сессию или создать новую"""
    course_id = request.query_params.get("course_id")
    course = Course.objects.get(pk=course_id)
    
    # Ищем незавершённую сессию
    session = LearningSession.objects.filter(
        user=request.user,
        course=course,
        finished_at__isnull=True
    ).first()
    
    # Если нет, создаём новую
    if not session:
        session = LearningSession.objects.create(
            user=request.user,
            course=course,
            current_stage="start"
        )
    
    return Response({
        "id": session.id,
        "current_stage": session.current_stage,
        "tasks_solved_count": session.tasks_solved_count,
        "tasks_correct_count": session.tasks_correct_count,
        "score_percent": session.score_percent,
    })
```

#### 3.3. Получение следующей задачи

Создайте эндпоинт для получения следующей задачи на основе алгоритма:

**Требования:**
- `GET /api/session/<id>/next_task/` — получить следующую задачу
- Использование адаптивного алгоритма для выбора сложности
- Исключение уже решённых задач
- Возврат задачи подходящей сложности

**Пример:**
```python
@action(detail=True, methods=["get"])
def next_task(self, request, pk=None):
    """Получить следующую задачу на основе адаптивного алгоритма"""
    session = LearningSession.objects.get(pk=pk, user=request.user)
    
    # Получаем уже решённые задачи
    solved_task_ids = TaskAttempt.objects.filter(
        session=session
    ).values_list('task_id', flat=True)
    
    # Определяем желаемую сложность (ваш алгоритм)
    target_difficulty = get_adaptive_difficulty(session)
    
    # Ищем задачу подходящей сложности
    next_task = Task.objects.filter(
        course=session.course,
        difficulty=target_difficulty
    ).exclude(id__in=solved_task_ids).first()
    
    if not next_task:
        # Если нет задач нужной сложности, берём любую нерешённую
        next_task = Task.objects.filter(
            course=session.course
        ).exclude(id__in=solved_task_ids).first()
    
    if not next_task:
        return Response({"detail": "Все задачи решены"}, status=404)
    
    return Response({
        "id": next_task.id,
        "title": next_task.title,
        "text": next_task.text,
        "difficulty": next_task.difficulty,
    })
```

```jsx
  return (
```

```jsx
      <div className="flex justify-between mb-2">
        {stages.map((stage, idx) => (
          <div
```

```python
class LearningSession(models.Model):
    """Сессия обучения"""
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    course = models.ForeignKey(Course, on_delete=models.CASCADE)
    
    current_stage = models.CharField(max_length=50, default="start")
    tasks_solved_count = models.PositiveIntegerField(default=0)
    tasks_correct_count = models.PositiveIntegerField(default=0)
    wrong_attempts_in_row = models.PositiveIntegerField(default=0)
    score_percent = models.FloatField(default=0)
    
    started_at = models.DateTimeField(auto_now_add=True)
    finished_at = models.DateTimeField(null=True, blank=True)
    
    def calculate_score(self):
        if self.tasks_solved_count > 0:
            self.score_percent = (
                self.tasks_correct_count / self.tasks_solved_count * 100
            )
        return self.score_percent

class TaskAttempt(models.Model):
    """Попытка решения задачи"""
    session = models.ForeignKey(LearningSession, on_delete=models.CASCADE)
    task = models.ForeignKey(Task, on_delete=models.CASCADE)
    answer_numeric = models.FloatField(null=True, blank=True)
    is_correct = models.BooleanField(default=False)
    created_at = models.DateTimeField(auto_now_add=True)
```

## Часто задаваемые вопросы

**Q: Как проверить, что алгоритм работает?**  
A: Создайте тестового пользователя, решите несколько задач (правильно и неправильно) и проверьте изменения в статистике и сложности.

**Q: Что делать, если все задачи решены?**  
A: Проверьте это в методе `next_task` и верните соответствующее сообщение или завершите сессию.

**Q: Как определить начальную сложность?**  
A: Можно использовать среднюю сложность (3) или дать ученику выбрать уровень сложности.

**Q: Нужно ли ограничивать количество попыток?**  
A: Это опционально, но можно добавить ограничение (например, максимум 3 попытки на задачу).

**Q: Как обрабатывать случай, когда нет задач нужной сложности?**  
A: Можно взять задачу ближайшей сложности или предложить задачи другой сложности.
